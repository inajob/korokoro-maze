# korokoro-maze

CadQuery を使用して、3Dプリント可能な迷路とその専用カバーを生成するスクリプト群です。
ランダムな迷路アルゴリズム（Recursive Backtracker）に基づいた迷路本体と、中身が見えるハニカム構造のカバーを出力します。

## 特徴

- **ランダム迷路生成**: 実行するたびに異なるパターンの迷路が生成されます。
- **ハニカムカバー**: 軽量かつ視認性の高い六角形メッシュのカバー。
- **組み立て設計**: 四隅に M3 ネジ用の穴（直径 4mm）が配置されており、本体とカバーを固定できます。
- **カスタマイズ可能**: セル数、セルサイズ、壁の厚み、高さなどをパラメータで簡単に変更できます。

## ファイル構成

- `maze.py`: 迷路本体の生成スクリプト。`maze.stl` を出力します。
- `maze-cover.py`: ハニカム構造カバーの生成スクリプト。`maze_cover.stl` を出力します。

## セットアップ

### 必須ライブラリ
以下のライブラリがインストールされている環境が必要です。

- [CadQuery](https://github.com/CadQuery/cadquery)
- [ocp-vscode](https://github.com/bernhard-42/vscode-ocp-cad-viewer) (VS Code でのプレビュー用)

```bash
pip install cadquery ocp-vscode
```

## 使い方

1.  VS Code で `maze.py` または `maze-cover.py` を開きます。
2.  OCP Viewer を起動します。
3.  スクリプトを実行すると、プレビューが表示され、カレントディレクトリに STL ファイルが保存されます。

## パラメータの調整

各スクリプト冒頭の `--- パラメータ ---` セクションを書き換えることで、形状をカスタマイズできます。

| パラメータ | 説明 | デフォルト値 |
| :--- | :--- | :--- |
| `rows`, `cols` | 迷路の行数と列数 | 8, 8 |
| `cell_size` | 1セルの大きさ (mm) | 19.0 |
| `outer_wall_thickness` | 外壁の厚み (mm) | 8.0 |
| `total_height_above_base` | 迷路の壁の高さ (mm) | 17.0 |
| `base_height` | 底面の厚み (mm) | 0.5 |
| `mesh_size` | カバーのハニカムの大きさ (mm) | 7.5 |
